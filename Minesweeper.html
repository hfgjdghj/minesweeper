Minesweeper

This program belongs to Ho Hin Cheung.

Introduction

Minesweeper is a classic game. Here, I try to write the minesweeper game by using Pascal as my programming language and lazarus as my source editor.

Details

I will show you how to use my program here

Main Features

  • First safe click Guranteed

    The game field is generated after the first click of the player !

  • Two control Methods

    Players can use mouse or keyboard to play the game (but not menu screen)

  • Auto settings

    The screen buffer size and the size of the console will be adjusted by the code in my program.

  • Well Designed interface

    In both control methods, the hightlighting of the selected block is enabled

  • Nice algorithms

    the movements and selecting of grids are designed and can cater lot of special cases

    ( Definitely Not perfect! That is why we always need to improve game designsÔºÅ )

  • Save and load saved game

    this program allows you to save your gameboard and play next time

  • Save statistics

    This program stores the wins and loses when you are playing the game

  • Different difficulties

    Three difficulties ( easy, normal, hard ) can be choosed

Kind Reminders when playing the game

  • Avoid fast mouse movement

    As the speed of detection of mouse position and mouse click is slow, some unwanted problems (may affect appearance but not the game algorithms) may appear when moving your mouse too fast. This problem cannot be solved by code.

  • Close the game by using the quit function on the menu

    the saving of statistics is done before closing

  • DO NOT DELETE GAME FILES

    The "savedboard.txt" and "stats.txt" are needed every time the program is opened. Editing or deleting the file will cause errors !

  • Never change the settings of the game console yourself

    The screen buffer size and the size of the console will be adjusted by the code in my program. If you change these settings on your own, terrible things will appear !

Starting Screen

This screen is controlled by the up and down arrow keys. Press enter to confirm your choice

  • start game - this will starts the game
  • load saved game - this will load the "savedboard.txt" from the directory where your .exe file is placed
  • statistics - this will let you see your wins, loses and total games
  • Quit - Quit the game

Choose Difficulty (after start game)(won't show up if load saved game)

This screen is controlled by the up and down arrow keys. Press enter to confirm your choice

There are 3 difficulties for players to choose

  • Easy - 10x10 grid , 10 mines
  • Normal - 18x14 grid, 40 mines
  • Hard - 22x18 grid , 99 mines

Choose Control method (After choose difficulty or after load saved game)

This screen is controlled by the up and down arrow keys. Press enter to confirm your choice

  • Mouse - the game will be controlled by moving the mouse and clicking the mouse buttons
  • Keyboard - the game will be controlled by pressing differect keyboard keys

Mouse Control

Move mouse to select. Left mouse button to open the grid or to click on the quit and save button. Right mouse button mark the grid as mine.

Features

  • Coloring - when your mouse moves on an unopened block, the block will be colored
  • Mines Left - Number of mines waiting for u to mark (will decrease whenever u mark a bomb)
  • Quit - the blue button on left hand corner, quit to menu screen
  • Save- Save your current game progress to "savedboard.txt"
  • Avoid Misclick - when u mark a grid as mine, it cannot be opened to avoid misclicking
  • Opening of Grids - the opening of grids are checked by recursion
  • Avoid Misclick - when u mark a grid as mine, it cannot be opened to avoid misclicking
  • Opening of Grids - the opening of grids are checked by recursion

Cautions:

Keyboard Control

arrow keys to move. q to quit game. s to save game. t to toggle between mines and empty grids and z , x to move to the previous or next mine or empty grid

Features

  • Coloring - the purple indicates which grid you are on
  • ToggleMine ( default as false, can be changed by pressing t )

    when true, the z and x keyboard keys will only go to the previous or next marked mines

    when false, the z and x keyboard keys will only go to the previous or next empty grids

    z is for previous, x is for next
